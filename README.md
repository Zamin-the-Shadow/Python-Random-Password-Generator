# 🔐 Random Password Generator
A simple and secure Random Password Generator built using Python.  
This project generates strong passwords using letters, numbers, and special characters to improve account security.

## 🚀 Features
- Generate random passwords instantly
- User-defined password length
- Includes:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Strong password generation logic
- Beginner-friendly Python project

## 🛠️ Technologies Used
- Python 3
- random module
- string module

## 📌 How It Works
1. User enters the desired password length
2. Program combines different character sets
3. A secure random password is generated
4. Password is displayed on the screen

## ▶️ Example Output
```bash
===== RANDOM PASSWORD GENERATOR =====
Enter password length: 10
Generated Password: A@7kL#2pQ!
```
